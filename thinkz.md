# my ZSH

### quick install
```
 git config --global http.proxy http://172.16.31.120:1081
 git config --global https.proxy http://172.16.31.120:1081

 git clone --depth=1 https://github.com/thinkz/myzsh ~/.oh-my-zsh
 cd ~/.oh-my-zsh
 git submodule update --init custom/themes/powerlevel10k
```

### quick config
```
bash ~/.oh-my-zsh/custom/_install.sh

cp custom/_zshrc ~/.zshrc
cp custom/_p10k.zsh ~/.p10k.zsh
```
